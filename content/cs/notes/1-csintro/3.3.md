## Simple-MIPS 控制指令

{{< callout type="info" >}}
本节继续介绍 **Simple-MIPS** 教学指令集中的控制流指令。
{{</ callout >}}

本节介绍几种基本的 Simple-MIPS 控制流指令，用于实现条件判断和循环：slt、sle、beqz、goto（跳转）。

### slt 指令

- **类型**：比较指令 (Set on Less Than)
- **功能**：比较两个寄存器的值，如果第一个小于第二个，则设置目标寄存器为 1，否则为 0。
- **语法**：`slt $rd, $rs, $rt`
- **说明**：
  - `$rs`：第一个源寄存器
  - `$rt`：第二个源寄存器
  - `$rd`：目标寄存器
- **逻辑**：`if ($rs < $rt) $rd = 1; else $rd = 0;`
- **示例**：`slt $t0, $s1, $s2`
  - 如果 `$s1` 小于 `$s2`，`$t0` 置为 1，否则置为 0。

### sle 指令

- **类型**：比较指令 (Set on Less Than or Equal)
- **功能**：比较两个寄存器的值，如果第一个小于等于第二个，则设置目标寄存器为 1，否则为 0。
- **语法**：`sle $rd, $rs, $rt`
- **说明**：这是一个**伪指令**，汇编器通常将其转换为 `slt` 相关的指令序列。
- **逻辑**：`if ($rs <= $rt) $rd = 1; else $rd = 0;`
- **示例**：`sle $t0, $s1, $s2`
  - 如果 `$s1` 小于或等于 `$s2`，`$t0` 置为 1。

### beqz 指令

- **类型**：条件分支指令 (Branch on Equal Zero)
- **功能**：如果寄存器的值为 0，则跳转到指定标签。
- **语法**：`beqz $rs, label`
- **说明**：
  - `$rs`：要检查的寄存器
  - `label`：跳转的目标标签
- **逻辑**：`if ($rs == 0) goto label;`
- **示例**：`beqz $t0, Loop`
  - 如果 `$t0` 为 0，程序跳转到 `Loop` 标签处执行。

### goto 指令

- **类型**：无条件跳转指令 (Unconditional Jump)
- **功能**：无条件跳转到指定地址。
- **语法**：`goto label`
- **说明**：
  - 在标准 MIPS 汇编中，无条件跳转指令通常是 `j` (Jump)。
  - 在 **Simple-MIPS** 中，为了更贴近高级语言的习惯，我们将其命名为 `goto`。
- **逻辑**：`PC = label`
- **示例**：`goto Exit`
  - 程序直接跳转到 `Exit` 标签处执行。
