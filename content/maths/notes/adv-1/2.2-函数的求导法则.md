## 一、基本初等函数的求导公式

以下公式是求导的基础，必须熟练掌握（$C$ 为常数，$a$ 为常数）。

| 函数 $y=f(x)$                                | 导函数 $y'=f'(x)$                      | 备注                        |
| :------------------------------------------- | :------------------------------------- | :-------------------------- |
| **常数函数** $y = C$                         | $y' = 0$                               |                             |
| **幂函数** $y = x^n$                         | $y' = n x^{n-1}$                       | (对任意实数 $n$ 成立)       |
| **指数函数** $y = a^x$                       | $y' = a^x \ln a$                       | ($a>0$ 且 $a \ne 1$)        |
| **自然指数函数** $y = e^x$                   | $y' = e^x$                             | (是上式的特例，$\ln e = 1$) |
| **对数函数** $y = \log_a x$                  | $y' = \frac{1}{x \ln a}$               | ($a>0$ 且 $a \ne 1, x>0$)   |
| **自然对数函数** $y = \ln x$                 | $y' = \frac{1}{x}$                     | (是上式的特例，$\ln e = 1$) |
| **正弦函数** $y = \sin x$                    | $y' = \cos x$                          |                             |
| **余弦函数** $y = \cos x$                    | $y' = -\sin x$                         |                             |
| **正切函数** $y = \tan x$                    | $y' = \sec^2 x = \frac{1}{\cos^2 x}$   |                             |
| **余切函数** $y = \cot x$                    | $y' = -\csc^2 x = -\frac{1}{\sin^2 x}$ |                             |
| **反正弦函数** $y = \arcsin x$               | $y' = \frac{1}{\sqrt{1-x^2}}$          |                             |
| **反余弦函数** $y = \arccos x$               | $y' = -\frac{1}{\sqrt{1-x^2}}$         |                             |
| **反正切函数** $y = \arctan x$               | $y' = \frac{1}{1+x^2}$                 |                             |
| **反余切函数** $y = \operatorname{arccot} x$ | $y' = -\frac{1}{1+x^2}$                |                             |

## 二、函数的四则运算求导法则

设 $u=u(x)$ 和 $v=v(x)$ 都是可导函数，则：

1. **常数与函数相乘法则**
   $$[C u(x)]' = C u'(x)$$
2. **函数和（差）的求导法则**
   $$[u(x) \pm v(x)]' = u'(x) \pm v'(x)$$
3. **函数积的求导法则**
   $$[u(x) v(x)]' = u'(x) v(x) + u(x) v'(x)$$
   (三个函数相乘：$[u v w]' = u' v w + u v' w + u v w'$)
4. **函数商的求导法则**
   $$\left[\frac{u(x)}{v(x)}\right]' = \frac{u'(x) v(x) - u(x) v'(x)}{[v(x)]^2} \quad (v(x) \ne 0)$$
   (特例：$\left[\frac{C}{v(x)}\right]' = -\frac{C v'(x)}{[v(x)]^2}$)

## 三、复合函数的求导法则（链式法则）

设函数 $y = f(u)$ 且 $u = g(x)$，则复合函数 $y = f(g(x))$ 的导数为：

$$\frac{\mathrm{d} y}{\mathrm{~d} x} = \frac{\mathrm{d} y}{\mathrm{~d} u} \cdot \frac{\mathrm{d} u}{\mathrm{~d} x}$$

或用 $f'(x)$ 记号表示为：

$$[f(g(x))]' = f'(g(x)) \cdot g'(x)$$

**推广：** 对于多重复合函数 $y = f(u)$，$u = g(v)$，$v = h(x)$，则：
$$\frac{\mathrm{d} y}{\mathrm{~d} x} = \frac{\mathrm{d} y}{\mathrm{~d} u} \cdot \frac{\mathrm{d} u}{\mathrm{~d} v} \cdot \frac{\mathrm{d} v}{\mathrm{~d} x}$$

## 四、其他常见函数的求导方法

### 1. 幂指函数的求导

幂指函数形如 $y = [u(x)]^{v(x)}$，必须使用**对数求导法**。

1. **转化：** $y = e^{v(x) \ln u(x)}$
2. **求导：**
   $$y' = [u(x)]^{v(x)} \left[ v'(x) \ln u(x) + \frac{v(x) u'(x)}{u(x)} \right]$$

### 2. 隐函数的求导

对于由方程 $F(x, y) = 0$ 所确定的隐函数 $y=y(x)$，其求导方法是：

1. 将方程 $F(x, y) = 0$ 对 $x$ 求导。
2. 将 $y$ 看作 $x$ 的函数，对所有含 $y$ 的项应用**复合函数求导法则**，即乘以 $y'$。
3. 解出求导后的方程，得到 $y'$。

**例如：** 对 $x^2 + y^2 = 1$ 求导：
$$[x^2]' + [y^2]' = [1]'$$
$$2x + 2y \cdot y' = 0$$
$$y' = -\frac{2x}{2y} = -\frac{x}{y}$$

### 3. 反函数的求导

设函数 $y = f(x)$ 严格单调且可导，其反函数为 $x = \phi(y)$，则反函数的导数为：

$$\frac{\mathrm{d} x}{\mathrm{~d} y} = \frac{1}{\frac{\mathrm{d} y}{\mathrm{~d} x}} \quad (\text{其中} \frac{\mathrm{d} y}{\mathrm{~d} x} \ne 0)$$

### 4. 参数方程所确定的函数的求导

设函数由参数方程 $\begin{cases} x = \phi(t) \\ y = \psi(t) \end{cases}$ 确定，其中 $\phi(t)$ 和 $\psi(t)$ 可导，且 $\phi'(t) \ne 0$，则：

$$\frac{\mathrm{d} y}{\mathrm{~d} x} = \frac{\frac{\mathrm{d} y}{\mathrm{~d} t}}{\frac{\mathrm{d} x}{\mathrm{~d} t}} = \frac{\psi'(t)}{\phi'(t)}$$
