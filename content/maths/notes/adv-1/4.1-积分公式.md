---
title: 4.1 积分公式
---

在以下积分公式中，$C$ 代表任意常数，$\alpha$ 为任意实数且 $\alpha \ne -1$， $a$ 和 $k$ 为非零常数。

## 一、基础函数积分公式

这些公式是微分学的逆运算，是积分计算的核心。

- $$\int 0 \, \mathrm{d} x = C$$

- $$\int k \, \mathrm{d} x = k x + C$$

- $$\int x^{\alpha} \, \mathrm{d} x = \frac{x^{\alpha+1}}{\alpha+1} + C \quad (\alpha \ne -1)$$

- $$\int \frac{1}{x} \, \mathrm{d} x = \ln |x| + C$$

- $$\int a^x \, \mathrm{d} x = \frac{a^x}{\ln a} + C \quad (a > 0, a \ne 1)$$

- $$\int e^x \, \mathrm{d} x = e^x + C$$

## 二、三角函数积分公式

- $$\int \cos x \, \mathrm{d} x = \sin x + C$$

- $$\int \sin x \, \mathrm{d} x = -\cos x + C$$

- $$\int \sec^2 x \, \mathrm{d} x = \tan x + C$$

- $$\int \csc^2 x \, \mathrm{d} x = -\cot x + C$$

- $$\int \sec x \tan x \, \mathrm{d} x = \sec x + C$$

- $$\int \csc x \cot x \, \mathrm{d} x = -\csc x + C$$

## 三、常用扩展积分公式

这些公式在涉及分式和三角函数时非常常用。

- $$\int \tan x \, \mathrm{d} x = -\ln |\cos x| + C$$

- $$\int \cot x \, \mathrm{d} x = \ln |\sin x| + C$$

- $$\int \sec x \, \mathrm{d} x = \ln |\sec x + \tan x| + C$$

- $$\int \csc x \, \mathrm{d} x = \ln |\csc x - \cot x| + C$$

## 四、反三角和复杂根式积分公式

这些积分结果通常用于有理分式和根式代换后的计算。

- $$\int \frac{1}{a^2 + x^2} \, \mathrm{d} x = \frac{1}{a} \arctan \left(\frac{x}{a}\right) + C$$

- $$\int \frac{1}{\sqrt{a^2 - x^2}} \, \mathrm{d} x = \arcsin \left(\frac{x}{a}\right) + C \quad (a > 0)$$

- $$\int \frac{1}{\sqrt{x^2 \pm a^2}} \, \mathrm{d} x = \ln |x + \sqrt{x^2 \pm a^2}| + C$$

- $$\int \frac{1}{x^2 - a^2} \, \mathrm{d} x = \frac{1}{2a} \ln \left| \frac{x - a}{x + a} \right| + C$$

## 五、特殊积分公式

这些公式通常是使用分部积分法或三角代换得到的，建议记住结果。

- $$\int \ln x \, \mathrm{d} x = x \ln x - x + C$$

- $$\int \arctan x \, \mathrm{d} x = x \arctan x - \frac{1}{2} \ln (1+x^2) + C$$

- $$\int \sqrt{a^2 - x^2} \, \mathrm{d} x = \frac{x}{2} \sqrt{a^2 - x^2} + \frac{a^2}{2} \arcsin \left(\frac{x}{a}\right) + C$$

- $$\int \sqrt{x^2 \pm a^2} \, \mathrm{d} x = \frac{x}{2} \sqrt{x^2 \pm a^2} \pm \frac{a^2}{2} \ln |x + \sqrt{x^2 \pm a^2}| + C$$
