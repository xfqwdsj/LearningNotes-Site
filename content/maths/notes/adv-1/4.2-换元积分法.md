---
title: 4.2 换元积分法
---

换元积分法（也称变量代换法）是求解不定积分的最主要方法之一。它的核心思想是：通过引入新的变量 $u$，将复杂函数的积分转化为简单函数 $u$ 的积分，从而利用基本积分表求解。

## 一、第一类换元法（凑微分法）

第一类换元法是复合函数求导法则的逆应用，适用于被积函数是某复合函数微分形式的情况。

### 1. 第一类换元法原理

设 $F(u)$ 是 $f(u)$ 的原函数，即 $F'(u) = f(u)$。如果 $u = \varphi(x)$ 可导，则根据复合函数求导的链式法则和不定积分的定义，有：

$$\int f(\varphi(x)) \varphi'(x) \, \mathrm{d} x = \int f(\varphi(x)) \, \mathrm{d} (\varphi(x)) = \int f(u) \, \mathrm{d} u = F(u) + C = F(\varphi(x)) + C$$

### 2. 核心技巧：凑微分

关键是将被积表达式 $f(\varphi(x)) \cdot \varphi'(x) \, \mathrm{d} x$ 中的 $\varphi'(x) \, \mathrm{d} x$ **凑成** $\mathrm{d}(\varphi(x))$。

| 常见凑微分形式                           | 凑微分结果                        |
| :--------------------------------------- | :-------------------------------- |
| $\mathrm{d} x$                           | $\mathrm{d} x$                    |
| $k \, \mathrm{d} x$                      | $\mathrm{d} (k x)$                |
| $x \, \mathrm{d} x$                      | $\frac{1}{2} \, \mathrm{d} (x^2)$ |
| $\sin x \, \mathrm{d} x$                 | $-\mathrm{d} (\cos x)$            |
| $\cos x \, \mathrm{d} x$                 | $\mathrm{d} (\sin x)$             |
| $\frac{1}{x} \, \mathrm{d} x$            | $\mathrm{d} (\ln \vert x\vert)$   |
| $e^x \, \mathrm{d} x$                    | $\mathrm{d} (e^x)$                |
| $\frac{1}{\sqrt{1-x^2}} \, \mathrm{d} x$ | $\mathrm{d} (\arcsin x)$          |
| $\frac{1}{1+x^2} \, \mathrm{d} x$        | $\mathrm{d} (\arctan x)$          |

#### 小技巧

将式子从不定积分符号内移入 $\mathrm{d}$，只需对它求原函数；将式子从 $\mathrm{d}$ 内移出不定积分符号，只需对它求导。$\mathrm{d}$ 中增减常数不影响结果。

### 3. 例题演示（线性替换）

当 $\varphi(x) = ax + b$ 时，有 $\mathrm{d}(ax+b) = a \, \mathrm{d} x$，所以 $\mathrm{d} x = \frac{1}{a} \mathrm{d}(ax+b)$。
$$\int \cos(3x+2) \, \mathrm{d} x = \int \cos(3x+2) \cdot \frac{1}{3} \, \mathrm{d} (3x+2)$$
$$\text{令 } u = 3x+2$$
$$= \frac{1}{3} \int \cos u \, \mathrm{d} u = \frac{1}{3} \sin u + C = \frac{1}{3} \sin(3x+2) + C$$

## 二、第二类换元法（代换）

第二类换元法是通过主动设置代换 $x = \psi(t)$，将复杂的被积函数形式替换掉。

### 1. 第二类换元法原理

设 $x = \psi(t)$ 是单调、可导的函数，且 $\psi'(t) \ne 0$。则：

$$\int f(x) \, \mathrm{d} x = \int f(\psi(t)) \psi'(t) \, \mathrm{d} t$$

求出右侧关于 $t$ 的原函数 $G(t) + C$ 后，必须将其代回原变量 $x$（即 $t = \psi^{-1}(x)$），得到 $G(\psi^{-1}(x)) + C$。

### 2. 核心代换类型

第二类换元法主要用于消除被积函数中的复杂根式。

#### A. 三角代换（用于消除 $\sqrt{a^2 - x^2}$ 型根式）

| 根式类型           | 换元公式       | 限制条件                                              | 积分结果形式                  |
| :----------------- | :------------- | :---------------------------------------------------- | :---------------------------- |
| $\sqrt{a^2 - x^2}$ | $x = a \sin t$ | $t \in [-\frac{\pi}{2}, \frac{\pi}{2}]$               | $\sqrt{a^2 - x^2} = a \cos t$ |
| $\sqrt{a^2 + x^2}$ | $x = a \tan t$ | $t \in (-\frac{\pi}{2}, \frac{\pi}{2})$               | $\sqrt{a^2 + x^2} = a \sec t$ |
| $\sqrt{x^2 - a^2}$ | $x = a \sec t$ | $t \in [0, \frac{\pi}{2}) \cup [\pi, \frac{3\pi}{2})$ | $\sqrt{x^2 - a^2} = a \tan t$ |

#### B. 根式代换（用于消除 $\sqrt[n]{ax+b}$ 型根式）

设 $u = \sqrt[n]{ax+b}$，则 $u^n = ax+b$。
$$\int f(\sqrt[n]{ax+b}) \, \mathrm{d} x \quad \xrightarrow{\text{令 } u = \sqrt[n]{ax+b}} \quad \int f(u) \cdot \frac{n}{a} u^{n-1} \, \mathrm{d} u$$

#### C. 万能代换（用于有理化三角函数）

$$\int R(\sin x, \cos x) \, \mathrm{d} x$$
$$\text{令} t = \tan\left(\frac{x}{2}\right) \quad \text{则} \sin x = \frac{2t}{1+t^2}, \cos x = \frac{1-t^2}{1+t^2}, \mathrm{d} x = \frac{2}{1+t^2} \, \mathrm{d} t$$

### 3. 例题演示（三角代换）

求 $\int \frac{1}{\sqrt{4 - x^2}} \, \mathrm{d} x$。

1. **换元：** 根式为 $\sqrt{a^2 - x^2}$ 型，其中 $a=2$。
   令 $x = 2 \sin t$，则 $\mathrm{d} x = 2 \cos t \, \mathrm{d} t$。
2. **替换：**
   $$\sqrt{4 - x^2} = \sqrt{4 - 4 \sin^2 t} = \sqrt{4 \cos^2 t} = 2 |\cos t|$$
   取 $t \in (-\frac{\pi}{2}, \frac{\pi}{2})$，则 $\cos t > 0$， $\sqrt{4 - x^2} = 2 \cos t$。
3. **积分：**
   $$\int \frac{2 \cos t}{2 \cos t} \, \mathrm{d} t = \int 1 \, \mathrm{d} t = t + C$$
4. **代回：**
   由 $x = 2 \sin t \implies t = \arcsin \left(\frac{x}{2}\right)$。
   $$\text{原式} = \arcsin \left(\frac{x}{2}\right) + C$$
