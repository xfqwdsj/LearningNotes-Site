---
title: 7.2 可分离变量的微分方程
---

## 一、定义与形式

如果一个一阶微分方程可以写成如下形式：
$$\frac{\mathrm{d}y}{\mathrm{d}x} = f(x)g(y)$$
或者
$$M_1(x)N_1(y) \, \mathrm{d}x + M_2(x)N_2(y) \, \mathrm{d}y = 0$$
即方程中的自变量 $x$ 和未知函数 $y$ 可以分别聚集在等号的两边，这种方程就称为**可分离变量的微分方程**。

## 二、解题步骤

解这类方程的核心思想是：**各找各妈，两边积分**。

1. **分离变量**：
   将所有含 $y$ 的项（包括 $\mathrm{d}y$）移到等号左边，所有含 $x$ 的项（包括 $\mathrm{d}x$）移到等号右边：
   $$\frac{1}{g(y)} \, \mathrm{d}y = f(x) \, \mathrm{d}x$$
   _(注意：需假设 $g(y) \neq 0$)_

2. **两边积分**：
   对等号两边同时进行不定积分：
   $$\int \frac{1}{g(y)} \, \mathrm{d}y = \int f(x) \, \mathrm{d}x$$

3. **整理成通解**：
   计算积分后得到：
   $$G(y) = F(x) + C$$
   如果题目给出了初始条件（如 $y\vert_{x=x_0} = y_0$），则代入求出常数 $C$ 的具体值，得到**特解**。

## 三、典型例题

### 例 1：求微分方程 $\frac{\mathrm{d}y}{\mathrm{d}x} = \frac{x}{y}$ 的通解

1. **分离变量**：
   将 $y$ 乘到左边，$\mathrm{d}x$ 乘到右边：
   $$y \, \mathrm{d}y = x \, \mathrm{d}x$$
2. **两边积分**：
   $$\int y \, \mathrm{d}y = \int x \, \mathrm{d}x$$
   $$\frac{1}{2}y^2 = \frac{1}{2}x^2 + C_1$$
3. **整理**：
   $$y^2 - x^2 = C \quad (\text{其中 } C = 2C_1)$$
   该通解在几何上表示一族**等轴双曲线**。

### 例 2：初值问题求解

求 $y' = 2xy$，满足条件 $y\vert_{x=0} = 1$。

1. **分离变量**：
   $$\frac{\mathrm{d}y}{y} = 2x \, \mathrm{d}x$$
2. **两边积分**：
   $$\int \frac{1}{y} \, \mathrm{d}y = \int 2x \, \mathrm{d}x$$
   $$\ln \vert y \vert = x^2 + C_1$$
3. **求通解**：
   $$\vert y \vert = e^{x^2 + C_1} = e^{C_1} \cdot e^{x^2}$$
   令 $C = \pm e^{C_1}$，得到 $y = C e^{x^2}$。
4. **代入初值**：
   当 $x=0$ 时 $y=1$：$1 = C \cdot e^0 \implies C = 1$。
5. **结论**：
   特解为 $y = e^{x^2}$。

## 四、技巧与注意事项

- **关于常数 $C$**：
  在积分过程中，两边都会产生常数，但习惯上只在右边写一个 $C$。有时为了整理方便，可以将 $C$ 写成 $\ln \vert C \vert$ 或 $\sin C$ 等形式。
- **遗漏解的情况**：
  在第一步分离变量（除以 $g(y)$）时，如果 $g(y_0) = 0$，那么常数函数 $y = y_0$ 可能是方程的一个解。在严格的考试中，需要检查这个常数解是否包含在最终的通解公式里。
- **微分形式的转换**：
  记住 $y' = \frac{\mathrm{d}y}{\mathrm{d}x}$，遇到 $y'$ 先换成这种形式。
