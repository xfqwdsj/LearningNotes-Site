---
title: 7.3 齐次方程
---

## 一、定义

如果一阶微分方程 $y' = f(x, y)$ 中的函数 $f(x, y)$ 可以写成自变量与因变量之比 $\frac{y}{x}$（或 $\frac{x}{y}$）的函数，即：
$$y' = \varphi\left( \frac{y}{x} \right)$$
则称此方程为**齐次方程**。

### 如何判断？

检查方程中每一项的**次数**。如果分子、分母中每一项的次数之和都相等，则该方程通常是齐次的。

- **例：** $y' = \frac{x^2 + y^2}{xy}$
  - 分子项 $x^2, y^2$ 均为 2 次；分母项 $xy$ 也是 $1+1=2$ 次。
  - 变形：$y' = \frac{x}{y} + \frac{y}{x}$，符合 $\varphi\left( \frac{y}{x} \right)$ 形式。

## 二、解题核心：换元法

解齐次方程的标准套路是引入中间变量 $u$。

1. **设变量**：
   令 $u = \frac{y}{x}$，则有 $y = ux$。
2. **求导数**（根据乘积求导法则）：
   $$\frac{\mathrm{d}y}{\mathrm{d}x} = u + x \frac{\mathrm{d}u}{\mathrm{d}x}$$
3. **代入原方程**：
   将 $y'$ 和 $\frac{y}{x}$ 替换掉，方程变为：
   $$u + x \frac{\mathrm{d}u}{\mathrm{d}x} = \varphi(u)$$
4. **分离变量并积分**：
   整理得到关于 $u$ 和 $x$ 的可分离变量方程：
   $$\frac{\mathrm{d}u}{\varphi(u) - u} = \frac{\mathrm{d}x}{x}$$
5. **变量还原**：
   算出结果后，务必将 $u$ 换回 $\frac{y}{x}$。

## 三、典型例题

### 例题：解微分方程 $(x^2 + y^2) \, \mathrm{d}x - xy \, \mathrm{d}y = 0$

**第一步：整理形式**
$$\frac{\mathrm{d}y}{\mathrm{d}x} = \frac{x^2 + y^2}{xy} = \frac{x}{y} + \frac{y}{x}$$

**第二步：换元**
令 $u = \frac{y}{x}$，则 $y' = u + x \frac{\mathrm{d}u}{\mathrm{d}x}$。
代入方程：
$$u + x \frac{\mathrm{d}u}{\mathrm{d}x} = \frac{1}{u} + u$$

**第三步：化简与分离变量**
左右两边消去 $u$：
$$x \frac{\mathrm{d}u}{\mathrm{d}x} = \frac{1}{u}$$
分离变量：
$$u \, \mathrm{d}u = \frac{\mathrm{d}x}{x}$$

**第四步：积分**
$$\int u \, \mathrm{d}u = \int \frac{\mathrm{d}x}{x}$$
$$\frac{1}{2} u^2 = \ln \vert x \vert + C_1$$
$$u^2 = 2 \ln \vert x \vert + C \quad (\text{其中 } C = 2C_1)$$

**第五步：还原变量**
将 $u = \frac{y}{x}$ 代入：
$$\left( \frac{y}{x} \right)^2 = 2 \ln \vert x \vert + C$$
$$y^2 = x^2 (2 \ln \vert x \vert + C)$$

## 四、延伸：可化为齐次方程的方程

有些方程形式如：
$$\frac{\mathrm{d}y}{\mathrm{d}x} = \frac{a_1 x + b_1 y + c_1}{a_2 x + b_2 y + c_2}$$
这类方程可以通过**平移坐标轴**的方法，消除常数项 $c_1, c_2$，从而将其转化为齐次方程。

- 通过解方程组 $\begin{cases} a_1 x + b_1 y + c_1 = 0 \\ a_2 x + b_2 y + c_2 = 0 \end{cases}$ 找到交点 $(h, k)$。
- 令 $X = x - h, Y = y - k$，方程即变为齐次形式。
