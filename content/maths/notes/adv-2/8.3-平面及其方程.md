---
title: 8.3 平面及其方程
---

在空间解析几何中，确定一个平面的位置通常需要两个要素：平面上的**一个点**和垂直于该平面的**一个向量**（法向量）。

## 一、平面的点法式方程

设平面上已知一点 $M_0(x_0, y_0, z_0)$，平面的一个**法向量**（垂直于平面的非零向量）为 $\mathbf{n} = (A, B, C)$。

对于平面上任意一点 $M(x, y, z)$，向量 $\overrightarrow{M_0M} = (x - x_0, y - y_0, z - z_0)$ 必垂直于 $\mathbf{n}$。
由数量积性质 $\mathbf{n} \cdot \overrightarrow{M_0M} = 0$ 可得：
$$A(x - x_0) + B(y - y_0) + C(z - z_0) = 0$$
这就是平面的**点法式方程**。

## 二、平面的一般方程

将点法式方程展开并整理，可以得到平面的一般形式：
$$Ax + By + Cz + D = 0$$
其中 $A, B, C$ 不全为零，且向量 $\mathbf{n} = (A, B, C)$ 即为此平面的一个**法向量**。

### 特殊情形讨论

- 若 $D = 0$：平面过原点 $(0, 0, 0)$。
- 若 $A = 0$：平面平行于 $x$ 轴（法向量垂直于 $x$ 轴）。
- 若 $A = B = 0$：平面平行于 $xy$ 坐标面（即垂直于 $z$ 轴）。

## 三、平面的截距式方程

若平面与三个坐标轴分别交于 $P(a, 0, 0)$、$Q(0, b, 0)$、$R(0, 0, c)$ 三点（且 $a, b, c \neq 0$），则平面方程可写为：
$$\frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1$$
其中 $a, b, c$ 称为平面在各坐标轴上的**截距**。

## 四、两平面的夹角

设平面 $\Pi_1$ 和 $\Pi_2$ 的法向量分别为 $\mathbf{n}_1 = (A_1, B_1, C_1)$ 和 $\mathbf{n}_2 = (A_2, B_2, C_2)$。
两平面的夹角 $\theta$（通常取锐角）定义为两法向量夹角的余弦值：
$$\cos \theta = \frac{\vert \mathbf{n}_1 \cdot \mathbf{n}_2 \vert}{\vert \mathbf{n}_1 \vert \vert \mathbf{n}_2 \vert} = \frac{\vert A_1 A_2 + B_1 B_2 + C_1 C_2 \vert}{\sqrt{A_1^2 + B_1^2 + C_1^2} \sqrt{A_2^2 + B_2^2 + C_2^2}}$$

### 两平面的位置关系

1. **垂直**：$\Pi_1 \perp \Pi_2 \iff \mathbf{n}_1 \cdot \mathbf{n}_2 = 0 \iff A_1 A_2 + B_1 B_2 + C_1 C_2 = 0$
2. **平行**：$\Pi_1 \parallel \Pi_2 \iff \mathbf{n}_1 \parallel \mathbf{n}_2 \iff \frac{A_1}{A_2} = \frac{B_1}{B_2} = \frac{C_1}{C_2}$

## 五、点到平面的距离

空间中一点 $P_0(x_0, y_0, z_0)$ 到平面 $Ax + By + Cz + D = 0$ 的距离 $d$ 为：
$$d = \frac{\vert A x_0 + B y_0 + C z_0 + D \vert}{\sqrt{A^2 + B^2 + C^2}}$$

## 六、平面方程的求法

求平面方程的核心通常是**找法向量 $\mathbf{n}$**。

- 如果已知平面过不在同一直线上的三点，可以利用其中两个向量的**向量积**（叉乘）来求 $\mathbf{n}$。
- 如果已知平面平行于两个已知向量 $\mathbf{a}$ 和 $\mathbf{b}$，则 $\mathbf{n} = \mathbf{a} \times \mathbf{b}$。
