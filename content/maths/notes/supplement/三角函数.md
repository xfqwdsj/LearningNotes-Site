## 一、三角函数的定义与基本关系

### 1. 角度与弧度

- **角度制：** 用 $360^{\circ}$ 表示一整周。
- **弧度制：** 用弧长与半径之比来衡量角的大小。 $360^{\circ} = 2\pi$ 弧度。换算关系：$180^{\circ} = \pi$ 弧度。

### 2. 六大基本三角函数定义（任意角 $\alpha$）

设角的终边经过点 $(x, y)$，该点到原点的距离 $r = \sqrt{x^2 + y^2} > 0$。

| 函数                     | 定义          | 倒数关系                    |
| :----------------------- | :------------ | :-------------------------- |
| **正弦** ($\sin \alpha$) | $\frac{y}{r}$ | $\csc \alpha = \frac{r}{y}$ |
| **余弦** ($\cos \alpha$) | $\frac{x}{r}$ | $\sec \alpha = \frac{r}{x}$ |
| **正切** ($\tan \alpha$) | $\frac{y}{x}$ | $\cot \alpha = \frac{x}{y}$ |

### 3. 基本恒等关系

- **平方关系：**

  $$\sin^2 \alpha + \cos^2 \alpha = 1$$

  $$\sec^2 \alpha - \tan^2 \alpha = 1$$

  $$\csc^2 \alpha - \cot^2 \alpha = 1$$

- **商数关系：**

  $$\tan \alpha = \frac{\sin \alpha}{\cos \alpha}$$

  $$\cot \alpha = \frac{\cos \alpha}{\sin \alpha}$$

## 二、诱导公式与基本性质

### 1. 诱导公式（常用）

将任意角的三角函数转化为锐角三角函数，口诀为 **“奇变偶不变，符号看象限”**。

| 角度                       | 正弦 $\sin$       | 余弦 $\cos$       |
| :------------------------- | :---------------- | :---------------- |
| $-\alpha$                  | $-\sin \alpha$    | $\cos \alpha$     |
| $\pi \pm \alpha$           | $\mp \sin \alpha$ | $-\cos \alpha$    |
| $\frac{\pi}{2} \pm \alpha$ | $\cos \alpha$     | $\mp \sin \alpha$ |

### 2. 周期性与奇偶性

| 函数 $f(x)$                      | 周期 $T$ | 奇偶性                                       | 备注 |
| :------------------------------- | :------- | :------------------------------------------- | :--- |
| $\sin x, \cos x, \sec x, \csc x$ | $2\pi$   | $\sin, \csc$ 为奇函数；$\cos, \sec$ 为偶函数 |      |
| $\tan x, \cot x$                 | $\pi$    | 均为奇函数                                   |      |

## 三、和差角、倍角与降幂公式（恒等变形）

### 1. 两角和差公式

- $$\sin(\alpha+\beta)=\sin\alpha\cos\beta+\cos\alpha\sin\beta$$

- $$\sin(\alpha-\beta)=\sin\alpha\cos\beta-\cos\alpha\sin\beta$$

- $$\cos(\alpha+\beta)=\cos\alpha\cos\beta-\sin\alpha\sin\beta$$

- $$\cos(\alpha-\beta)=\cos\alpha\cos\beta+\sin\alpha\sin\beta$$

- $$\tan(\alpha+\beta)=\dfrac{\tan\alpha+\tan\beta}{1-\tan\alpha\tan\beta}\quad(\cos\alpha\cos\beta\neq0)$$

- $$\tan(\alpha-\beta)=\dfrac{\tan\alpha-\tan\beta}{1+\tan\alpha\tan\beta}\quad(\cos\alpha\cos\beta\neq0)$$

### 2. 和差化积公式

- $$\sin\alpha+\sin\beta=2\sin\!\left(\dfrac{\alpha+\beta}{2}\right)\cos\!\left(\dfrac{\alpha-\beta}{2}\right)$$

- $$\sin\alpha-\sin\beta=2\cos\!\left(\dfrac{\alpha+\beta}{2}\right)\sin\!\left(\dfrac{\alpha-\beta}{2}\right)$$

- $$\cos\alpha+\cos\beta=2\cos\!\left(\dfrac{\alpha+\beta}{2}\right)\cos\!\left(\dfrac{\alpha-\beta}{2}\right)$$

- $$\cos\alpha-\cos\beta=-2\sin\!\left(\dfrac{\alpha+\beta}{2}\right)\sin\!\left(\dfrac{\alpha-\beta}{2}\right)$$

### 3. 积化和差公式

- $$\sin\alpha\cos\beta=\dfrac{1}{2}\big[\sin(\alpha+\beta)+\sin(\alpha-\beta)\big]$$

- $$\cos\alpha\sin\beta=\dfrac{1}{2}\big[\sin(\alpha+\beta)-\sin(\alpha-\beta)\big]$$

- $$\cos\alpha\cos\beta=\dfrac{1}{2}\big[\cos(\alpha+\beta)+\cos(\alpha-\beta)\big]$$

- $$\sin\alpha\sin\beta=\dfrac{1}{2}\big[\cos(\alpha-\beta)-\cos(\alpha+\beta)\big]$$

### 4. 倍角公式

- $$\sin(2\alpha)=2\sin\alpha\cos\alpha$$

- $$\cos(2\alpha)=\cos^2\alpha-\sin^2\alpha=2\cos^2\alpha-1=1-2\sin^2\alpha$$

- $$\tan(2\alpha)=\dfrac{2\tan\alpha}{1-\tan^2\alpha}\quad(\cos\alpha\neq0,\ 1-\tan^2\alpha\neq0)$$

### 5. 半角公式

- $$\sin\dfrac{\alpha}{2}=\pm\sqrt{\dfrac{1-\cos\alpha}{2}}\quad(\text{符号取决于 }\dfrac{\alpha}{2}\text{ 所在象限})$$

- $$\cos\dfrac{\alpha}{2}=\pm\sqrt{\dfrac{1+\cos\alpha}{2}}\quad(\text{符号取决于 }\dfrac{\alpha}{2}\text{ 所在象限})$$

- $$\tan\dfrac{\alpha}{2}=\pm\sqrt{\dfrac{1-\cos\alpha}{1+\cos\alpha}}=\dfrac{\sin\alpha}{1+\cos\alpha}=\dfrac{1-\cos\alpha}{\sin\alpha}$$

### 6. 降幂公式

- $$\sin^2\alpha=\dfrac{1-\cos(2\alpha)}{2},\qquad \cos^2\alpha=\dfrac{1+\cos(2\alpha)}{2}$$

- $$\sin^3\alpha=\dfrac{3\sin\alpha-\sin(3\alpha)}{4},\qquad \cos^3\alpha=\dfrac{3\cos\alpha+\cos(3\alpha)}{4}$$

- $$\sin^4\alpha=\dfrac{3-4\cos(2\alpha)+\cos(4\alpha)}{8},\qquad \cos^4\alpha=\dfrac{3+4\cos(2\alpha)+\cos(4\alpha)}{8}$$

- 需要更多幂次的降幂可用复指数或倍角展开（例如用傅里叶系数或复数欧拉公式进行推广）。

## 四、反三角函数

### 1. 反三角函数定义（主值分支）

| 函数            | 定义域               | 值域                                         |
| :-------------- | :------------------- | :------------------------------------------- |
| $y = \arcsin x$ | $[-1, 1]$            | $\left[-\frac{\pi}{2}, \frac{\pi}{2}\right]$ |
| $y = \arccos x$ | $[-1, 1]$            | $[0, \pi]$                                   |
| $y = \arctan x$ | $(-\infty, +\infty)$ | $\left(-\frac{\pi}{2}, \frac{\pi}{2}\right)$ |

### 2. 基本关系

$$\arcsin x + \arccos x = \frac{\pi}{2} \quad (\vert x \vert \le 1)$$

## 五、导数与不定积分（高等数学应用）

### 1. 基本三角函数及其导数

| 函数 $f(x)$ | 导函数 $f'(x)$                    |
| :---------- | :-------------------------------- |
| $\sin x$    | $\cos x$                          |
| $\cos x$    | $-\sin x$                         |
| $\tan x$    | $\sec^2 x = \frac{1}{\cos^2 x}$   |
| $\cot x$    | $-\csc^2 x = -\frac{1}{\sin^2 x}$ |
| $\sec x$    | $\sec x \tan x$                   |
| $\csc x$    | $-\csc x \cot x$                  |

### 2. 反三角函数及其导数

| 函数 $f(x)$               | 导函数 $f'(x)$            |
| :------------------------ | :------------------------ |
| $\arcsin x$               | $\frac{1}{\sqrt{1-x^2}}$  |
| $\arccos x$               | $-\frac{1}{\sqrt{1-x^2}}$ |
| $\arctan x$               | $\frac{1}{1+x^2}$         |
| $\operatorname{arccot} x$ | $-\frac{1}{1+x^2}$        |

### 3. 三角函数的不定积分（基本积分表）

| 积分 $\int f(x) \, \mathrm{d} x$ | 结果                                | 备注                          |
| :------------------------------- | :---------------------------------- | :---------------------------- |
| $\int \sin x \, \mathrm{d} x$    | $-\cos x + C$                       |                               |
| $\int \cos x \, \mathrm{d} x$    | $\sin x + C$                        |                               |
| $\int \sec^2 x \, \mathrm{d} x$  | $\tan x + C$                        |                               |
| $\int \csc^2 x \, \mathrm{d} x$  | $-\cot x + C$                       |                               |
| $\int \tan x \, \mathrm{d} x$    | $-\ln \vert\cos x\vert + C$         | (凑微分 $\mathrm{d}(\cos x)$) |
| $\int \cot x \, \mathrm{d} x$    | $\ln \vert\sin x\vert + C$          | (凑微分 $\mathrm{d}(\sin x)$) |
| $\int \sec x \, \mathrm{d} x$    | $\ln \vert\sec x + \tan x\vert + C$ |                               |
| $\int \csc x \, \mathrm{d} x$    | $\ln \vert\csc x - \cot x\vert + C$ |                               |
